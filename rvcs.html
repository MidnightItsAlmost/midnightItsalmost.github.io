The Regulated Vent_Control_System (RVCS)

 

FIELD OF INVENTION

The invention relates to controlling the temperature of each room, by modifying the vent cover registers and using the existing central air conditioning and heating duct systems.

 

BACKGROUND OF THE INVENTION

Modifies the central air conditioning and heating duct systems vent cover registers so that they may open and close automatically. Each vent cover register will give the user the ability to adjust the temperature of each room.

 

SUMMARY OF THE INVENTION

The opening and closing of the vent cover registers can be regulated by a control unit, the control unit is mounted on the wall, as any standard thermostat in existing central air conditioning and heating systems today. The control unit has a thermostat and a control switch. The control switch has the settings Partial or Fullï¿½. The control switch changes how the central air conditioning and heating system is shut off.

 

All connecting wires can be laid inside or outside the central air conditioning and heating duct system. One vent cover register may be connected to another in sequence, or stemming from the control unit.

 

The vent cover registers can be designed to fit any of todays conventional central air conditioning and heating system duct openings, in residential or commercial use. Each vent cover register houses its own thermostat, a blade control motor, PCB, and adjustable dial (adjustable dial on model 1 only). The vent cover registers temperature setting, is received from the control unit or by the adjustable dial on model 1 which can be set to any specified temperature, when not connected to a control unit. When the vent cover registers inferred thermostat reaches the set temperature the vent cover register will close on its own. While the vent cover register is open, it sends a signal back to the control unit of being open, if connected to a control unit.

 

If the control unit is set on the Fullï¿½ setting, then while any one vent cover register is sending the signal of being open to the control unit, the control unit will keep the air and heating system on. When no vent cover register is sending the signal of being open, the control unit shuts off the system. If the control unit is set on the Partialï¿½ setting, then any signal being sent from a vent cover register of being open is ignored. On the Partialï¿½ setting, once the control units own internal thermostat reaches the set temperature, it shuts off the system.

 

In addition to this RVCS, the vent cover registers may be linked to a personal computer. With this addition to the RCVS it maybe accessed by way of the internet. This would allow the user to modify the control units programming (control unit model 2 only) from any PC.

 

RVCS Control unit model 1

This control unit model, works identically like todays standard thermostats. In which the blower fan and AC compressor or Heating coil and are turned on and off. The addition to the existing thermostats of today is the units trigger (the mercury, coil spring, or like temperature switch) may be diverted to the thermostats on RVCS vent cover registers (FULL setting). The Full setting is not limited to having all vent cover registers being the RVCS type. If non-RVCS vent cover registers are used in conjunction with RVCS vent cover registers, then the control unit will be regulated by the RVCS vent cover registers, used while in the Fullï¿½ setting. If the control unit is set on the Partialï¿½ setting the signal from RVCS vent cover registers of being open is ignored, and the control unit uses it own internal mercury, coil spring or like thermostat switch.

 

RVCS Control unit model 2

GENERAL NOTE: THE RVCS CONTROL UNIT MODEL 2, MAY HAVE MANUAL SWITCHES, MENU NAVIGATING BUTTONS THAT ARE USED WITH THE LED DISPLAY SCREEN, OR COMPLETELY OPERATED BY AN LED TOUCH ACTIVATED SCREEN.

 

This control unit model has a Partial or Full setting switch as in control unit model 1. The Partial setting in this model will limit its programming ability. In the Full setting this model has the ability to be programmed to identify the rooms that the RVCS vent cover register(s) are in. Its programmed by first indicating the RVCS identification number, then the user may name the location of one or more RVCS vent cover registers to his/hers own choosing. Once the RVCS vent cover registers are Identified, the user may indicate the preferred temperature for that room or that single RVCS vent cover register. The user may also program in a timer setting, that enables any room or rooms, or single RVCS vent cover register, to be operational for any duration of time, be it days, weeks, or months.

 

This model may also be linked to a Personal Computer. The benefit of linking the control unit to a PC is that its programming maybe modified over the internet.

 

OPERATIONS RVCS CONTROL UNIT model 2

When the RVCS control unit model 2 is turned on for the first time, the control unit checks for any installed RVCS vent cover registers connected to the system. Then the control unit determines the identification number of all RVCS vent cover registers. The control unit will display the amount of vent cover registers sharing the same identification number (It is not necessary for a vent cover register to share an identification number). The control unit will also check for any group of vent cover registers sharing the same identification number and verify that only one in this group is set to be the Masterï¿½. Once the amount of vent cover registers and their identification numbers are determined the amount is displayed along side of each identification number. The user is then asked if they wish to give the first identification number an alpha and/or numerical label. Examples: Master Bedroom, Executive Boardroom, or Room 304. After the label decision has been made then the control unit asks the user to give a set temperature for that label which indicates a room or rooms. This continues until all vent cover registers are labeled, and a set temperature has been given.

 

While the control is in operation, the temperatures of the vent cover registers are monitored continually. The control unit calculates the over all temperature to determine if it should be colder or warmer by the majority of temperatures being sent from any vent cover registers sending the signal of being open in comparison to their preset user temperature setting. This is how the control determines if the system should be set to Cool or Heat. Should there be a vent cover register be open and is set to 75 degrees and the temperature being received by that vent cover registers inferred thermostat is slightly colder let's say 74 degrees and the control unit has found that the majority of rooms require Cooling. This vent cover register(s) will close by a command from the control unit before cooling the remaining rooms. Vent cover registers, model 1 have a backup system for opening and closing, for this explanation see "Operations of RVCS vent cover register model 1".

 

This control unit model 2 also has a "Partial" setting. If this control unit is set the "Partial" setting, the control unit uses it own internal thermostat to determine when to turn the blower fan and AC compressor or heating coil, on and off as well as determining if the temperature requires heating or cooling.

 

RVCS Vent cover register model 1

This vent cover register, model houses its own standard and inferred thermostats, a blade control motor, PCB, and adjustable dial. The adjustable dial may be set to within any suitable temperature range. While the vent cover register is open a signal is sent back to the connected RVCS control unit models 1 or 2, of being open. Once the vent cover register reaches its user set temperature the vent cover register will close by way of its blade control motor. When the vent closes the vent cover register no longer sends a signal of being open to the connect control unit.

 

OPERATIONS OF RVCS VENT COVER REGISTER MODEL 1

The RVCS vent cover register model 1, can be operated independently, or in conjunction with other RVCS vent cover registers models 1 or 2 without the aid of a control unit. If any vent cover register either model 1 or 2 is not connected to a control unit then a power supply is needed. The power supply can be given by way of a DC battery located on the back of the vent cover register, or a AC/DC converter plugged in to the Masterï¿½ vent cover register, and a standard electrical outlet. Should an AC/DC converter be used, then all connecting vent cover registers may share this power source. This AC/DC converter power source supply should be limited to approximately 4 vent cover registers.

 

RVCS vent cover registers used without the aid of a Control unit:

When the vent cover register is used independently, then once the temperature setting is reached the vent cover register closes. The vent cover register determines when it is necessary to open and close by it's user set temperature dial, the inferred thermostat, and a standard coil or mercury thermostat located on the back of the vent cover register. Should the vent cover register be open and the user supplied temperature setting is 75 degrees, and the temperature being received from the inferred thermostat is 74 degrees, as well as the AC system operating in cooling mode (blowing cold air through the duct system). Then once the air flows to the standard thermostat on the back of the vent cover register, the thermostat will find that the air flow is cold, in which case the vent cover register closes, preventing it's location from getting even colder. This is the backup system as stated in "RVCS control unit model 2 Operations".

 

When vent cover registers models 1 or 2, are used in conjunction with other RVCS vent cover registers models 1 or 2 without the aid of a control unit. Then there must be at least one vent cover register model 1 connected. Should there be only one vent cover register model 1 connected to one or more vent cover register model 2s, then the vent cover register model 1 controls the operations of all other connected vent cover registers that are model 2. When the vent cover register model 1 has determined that it needs to open or close it will also command the other vent cover registers that are model 2, to do the same. With this configuration of the RVCS the temperatures being received from the vent cover registers that are model 2, are ignored.

 

When two or more vent cover registers that are model 1 are used in conjunction, the "Slave" or "Master" switch must be set on each vent cover register that is model 1. When a vent cover register model 1 is connected to other vent cover registers that are model 1 and set to the "Slave" setting its temperature setting and thermostats signals are blocked and ignored by the connected vent cover register model 1 that is set to the "Master" setting.

 

RVCS vent cover registers used with and are connected to a Control unit:

When any RVCS vent cover registers either model 1 or 2, are connected to a control unit models 1 or 2, their power supply is received from that control unit.

 

When used with Control unit model 1:

When vent cover registers are used with control unit model 1, all vent cover registers that are not linked to other vent cover registers, must be model 1. A single vent cover register model 1 my still control any vent cover registers that are model 2. The control unit model 1 will not control a vent cover register model 2. The vent cover register model 1 opens and closes independently. While the vent cover register is open, it sends a signal of being open back to the control unit. For more information see "RVCS control unit model 1 Operations".

 

Alternate 1 design function when used with Control unit model 2:

When vent cover registers model 1 are used while being connected to the control unit model 2, and have the same user set identification number. Then only one vent cover register model 1 having this shared identification number maybe set to the "Master" setting. All other vent cover registers that are model 1 and having this shared identification number must be set to the "Slave" setting.

 

Alternate 2 design function when used with Control unit model 2:

When vent cover registers model 1 are used while being connected to the control unit model 2, and have the same user set identification number. Then the control unit takes all temperatures readings and finds the average temperature. This is the temperature the control unit model 2 uses to control the opening and closing of any set of vents covers having a shared identification number.

 

RVCS Vent cover register model 2

This vent cover register model has all the same components as vent cover register model 1, with the exception that this model has no adjustable dial. When is vent cover register model is use without any connect vent cover register model 1s, then it may only be used with RVCS control unit 2, and is solely controlled by it. One of this vent cover register must also be set as the Masterï¿½ in any set sharing the same identification number. As stated under design functions as described in RVCS vent cover register model 1.

Should there be only one vent cover register model 1 connected to one or more vent cover register model 2s, then the vent cover register model 1 controls the operations of all other connected vent cover registers that are model 2. When the vent cover register model 1 has determined that it needs to open or close it will also command the other vent cover registers that are model 2, to do the same. With this configuration of the RVCS the temperatures being received from the vent cover registers that are model 2, are ignored.

 

When two or more vent cover registers that are model 1 are used in conjunction, the "Slave" or "Master" switch must be set on each vent cover register that is model 1. When a vent cover register model 1 is connected to other vent cover registers that are model 1 and set to the "Slave" setting its temperature setting and thermostats signals are blocked and ignored by the connected vent cover register model 1 that is set to the "Master" setting.

 

RVCS vent cover registers used with and are connected to a Control unit:

When any RVCS vent cover registers either model 1 or 2, are connected to a control unit models 1 or 2, their power supply is received from that control unit.

 

When used with Control unit model 1:

When vent cover registers are used with control unit model 1, all vent cover registers that are not linked to other vent cover registers, must be model 1. A single vent cover register model 1 my still control any vent cover registers that are model 2. The control unit model 1 will not control a vent cover register model 2. The vent cover register model 1 opens and closes independently. While the vent cover register is open, it sends a signal of being open back to the control unit. For more information see "RVCS control unit model 1 Operations".

 

Alternate 1 design function when used with Control unit model 2:

When vent cover registers model 1 are used while being connected to the control unit model 2, and have the same user set identification number. Then only one vent cover register model 1 having this shared identification number maybe set to the "Master" setting. All other vent cover registers that are model 1 and having this shared identification number must be set to the "Slave" setting.

 

Alternate 2 design function when used with Control unit model 2:

When vent cover registers model 1 are used while being connected to the control unit model 2, and have the same user set identification number. Then the control unit takes all temperatures readings and finds the average temperature. This is the temperature the control unit model 2 uses to control the opening and closing of any set of vents covers having a shared identification number.
